# 桌面应用测试简介

> 从**Katalon Studio 版本 7.0.0**开始，您可以在 Windows 桌面应用程序上执行测试。

Katalon Studio 完全支持在以下平台上编写的桌面应用程序的自动化测试：通用 Windows 平台 (UWP)、Windows 窗体 (WinForms)、Windows 演示基础(WPF) 和经典 Windows (Win32)。
